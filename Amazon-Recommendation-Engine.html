Amazon Recommendation Engine

Project Overview

This project implements different recommendation engine models for Amazon product ratings dataset, focusing on popularity-based and collaborative filtering techniques. These techniques can help e-commerce platforms recommend suitable products for their users based on user's preferences or general product trends.

Core Functionalities

  • Data Loading and Sampling:
    • Loads the dataset from `ratings_Electronics.csv` using Pandas.
    • Takes a random 20% sample from the dataset to handle memory issues.
    • Removes timestamp column as it's not used.
  • Data Exploration and Cleaning:
    • Analyzes the distribution of ratings.
    • Calculates basic statistics of rating like mean, min, max etc.
    • Handles missing values in dataset.
    • Removes duplicate records in dataset.
  • Popularity Based Recommendation:
    • Filters products with at least 50 ratings.
    • Ranks products based on the number of ratings they've received.
    • Identifies the most popular products based on the number of ratings.
  • Item-Item Collaborative Filtering:
    • Employs item-based collaborative filtering using the KNNWithMeans algorithm from the Surprise library.
    • Calculates RMSE to measure the performance of the model.
  • Matrix Factorization:
    • Converts the data into a user-item rating matrix.
    • Uses Singular Value Decomposition (SVD) for dimensionality reduction.
    • Computes the correlation matrix for the products.
    • Recommends top 20 products based on their correlation.

Technical Details

  • Programming Language: Python 3.11.
  • Libraries/Modules Used:
    • pandas: Used for data manipulation and analysis, particularly with DataFrames.
    • numpy: Used for numerical operations and handling arrays.
    • seaborn and matplotlib: Used for creating visualizations.
    • surprise: Used for implementing collaborative filtering algorithms.
    • sklearn.decomposition: Used for performing Singular Value Decomposition.
  • Data Storage: CSV (`ratings_Electronics.csv`).
  • Machine Learning Techniques:
    • Popularity Based Filtering
    • Item-Item Collaborative Filtering
    • Matrix Factorization using Singular Value Decomposition

How to Use

  1. Install Required Libraries:

    Make sure you have all the necessary libraries installed. You can install them using pip:

    pip install pandas numpy seaborn matplotlib scikit-surprise
  2. Download the dataset:

    Make sure you have downloaded the data set (ratings_Electronics.csv)

  3. Prepare the Data:

    Ensure the CSV data file (`ratings_Electronics.csv`) is in the same directory as your Python script or Jupyter notebook.

  4. Run the Script:

    Execute the Python script or Jupyter Notebook containing the provided code.

  5. Review the Output:

    Examine the output, which will show the top products from popularity based recommendation, results of item-item collabrative filtering and also recommendations for particular product id's using SVD approach.
