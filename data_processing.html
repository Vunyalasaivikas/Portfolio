Employee Data Management System

Project Overview

This project implements an employee data management system that allows you to perform basic database operations as well as filter and sort data. This system integrates SQLite databases and uses Pandas to export data to an excel file.

Core Functionalities

  • File Handling:
    • Identifies files named with 'employee' in the current directory.
    • Locates and parses a JSON file (`employee.json`).
  • Data Extraction:
    • Extracts data from the JSON file to get the employee details.
    • Prints all the employee data.
    • Prints specific employee details when ID is provided.
  • Database Interaction:
    • Creates an SQLite database named data.db to store the employee data.
    • Creates an employee table in SQLite database based on the JSON data.
    • Inserts all the employee information from json to the SQLite table.
    • Reads all employee information from the SQLite database table.
  • Data Manipulation and Export:
    • Loads all the information from the SQLite database to Pandas Dataframe.
    • Adds a 'bonus percent' column to each employee, calculated based on the vaccination status.
    • Exports the final DataFrame with bonuses to an excel file (`employee.xlsx`).

Technical Details

  • Programming Language: Python 3.11.
  • Libraries/Modules Used:
    • os: Used for interacting with the operating system, such as listing files in a directory.
    • json: Used for parsing JSON data from a file.
    • sqlite3: Used for creating and interacting with SQLite databases.
    • pandas: Used for creating and manipulating DataFrames and exporting data to Excel files.
  • Data Storage: SQLite Database (`data.db`).
  • Data Format: JSON (`employee.json`) and Excel (`employee.xlsx`).

This project demonstrates the end-to-end process of reading, manipulating, and storing data using Python. From initial data parsing, database storage to final data transformation and export to a new file, this system showcases comprehensive data management.
