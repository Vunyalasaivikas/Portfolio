SQL Capstone Project

Project Overview

This project involves performing various analytical queries on NFT sales data using SQL. The data includes information about CryptoPunk sales, such as sale prices, dates, buyer addresses, and seller addresses. The goal was to extract meaningful insights from this data using complex SQL queries.

Core Functionalities

  • Data Analysis:
    • Performs in depth analysis of NFT sales data using SQL.
    • Extracts meaningful patterns and business-oriented insights.
  • Analytical Queries:
    • Determines the total number of sales within a specified date range.
    • Identifies the top 5 most expensive transactions based on USD price.
    • Calculates the moving average of USD price over the last 50 transactions.
    • Determines each NFT's average sales price, sorted in descending order.
    • Provides analysis of sales by each day of the week and their average price in ETH, ordered by the number of transactions.
    • Constructs a descriptive summary of each transaction with all the relevant details.
  • Advanced SQL Features:
    • Creates a database view to filter the sales from a given address.
    • Categorizes ETH price ranges, creating a histogram to illustrate price frequency.
    • Generates a unified query that shows the highest and lowest purchase prices for each NFT.
    • Determines the NFT with the highest sales count and the maximum sale price for a given month and year.
    • Calculates the total sales volume on a monthly basis.
    • Determines number of sales made by a given wallet address as a buyer, seller and total transactions.
    • Builds a robust value calculator that filters out outliers in daily sales and calculates a representative average price.
    • Determines the overall profitability (or loss) of each wallet, ordered by the most profitable.

Technical Details

  • Programming Language: SQL.
  • Database: SQLite or other SQL database
  • Data Storage: CSV (`cryptopunks.csv`)
  • Output: Query results and data insights.

How to Use

  1. Prepare the Data:

    Ensure the CSV data file (`cryptopunks.csv`) is loaded into SQL database.

  2. Execute the SQL Script:

    Execute the SQL script or code containing the queries. You can use an SQL client or other tool.

  3. Analyze Results:

    Examine the output generated from the SQL queries, which would provide information and insights about the NFT sales data.
