Credit Card Fraud Detection

Project Overview

This project focuses on building a machine learning model to detect fraudulent credit card transactions. The project includes data preprocessing, exploratory data analysis, model training, hyperparameter tuning, and evaluation steps. A random forest classifier is used in this project to make predictions.

Core Functionalities

  • Data Handling:
    • Loads the dataset, explores its structure, and understands features.
    • Checks for missing values and descriptive statistics.
  • Data Preprocessing:
    • Converts the time variable into a day format.
    • Performs data transformation of the features.
    • Implements undersampling to balance the skewed data.
  • Model Training and Evaluation:
    • Splits the dataset into training and testing sets.
    • Trains, evaluates, and compares various models like logistic regression, decision tree, random forest, svm and k nearest neighbours.
    • Selects the Random forest model as the main model.
  • Model Optimization and Validation:
    • Uses GridSearchCV to tune the parameters of Random Forest model.
    • Tests the performance of the model using metrics like precision, recall, f1-score and ROC curve and confusion matrix.
  • Random Data and Prediction:
    • Generates random data and makes predictions.
    • Stores the predicted data in an excel sheet.
  • Pipeline Creation:
    • Creates a pipeline for scaling the data and prediction.

Technical Details

  • Programming Language: Python 3.11.
  • Libraries/Modules Used:
    • pandas: Used for data manipulation and analysis, particularly with DataFrames.
    • numpy: Used for numerical computations and handling NaN values.
    • scikit-learn: Used for model training, evaluation, and pipeline creation.
    • imbalanced-learn: Used for balancing the imbalanced dataset
    • openpyxl: Used for creating and writing to Excel files.
  • Data Storage: CSV (`creditcard.csv`), Excel Sheet
  • Output: Trained model, model performance metrics, Excel sheet containing the random data and predicted output.

How to Use

  1. Install Required Libraries:

    Make sure you have all the necessary libraries installed. You can install them using pip:

    pip install pandas numpy scikit-learn imbalanced-learn openpyxl
  2. Prepare the Data:

    Ensure the CSV data file (`creditcard.csv`) is in the same directory as your Python script or provide the correct file path.

  3. Run the Script:

    Execute the Python script or Jupyter Notebook containing the provided code.

  4. Review the Output:

    Review the generated metrics for model performance and the output excel sheet which has the random generated data and prediction
