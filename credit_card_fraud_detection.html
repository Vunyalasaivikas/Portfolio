Credit Card Fraud Detection

Project Overview

This project focuses on building a machine learning model to detect fraudulent credit card transactions. The project includes data preprocessing, exploratory data analysis, model training, hyperparameter tuning, and evaluation steps. A random forest classifier is used in this project to make predictions.

Core Functionalities

  • Data Handling:
    • Loads the credit card transaction dataset.
    • Explores the dataset's structure and features.
    • Checks for any missing values and calculates descriptive statistics.
  • Data Preprocessing:
    • Converts the 'Time' feature to a day-based representation.
    • Applies necessary feature transformations for model compatibility.
    • Implements undersampling techniques to balance the class distribution.
  • Model Training and Evaluation:
    • Splits the data into training and testing sets.
    • Trains and evaluates multiple models, including Logistic Regression, Decision Tree, Random Forest, SVM, and KNN.
    • Selects the Random Forest model as the primary model.
  • Model Tuning and Validation:
    • Optimizes the Random Forest model's hyperparameters using GridSearchCV.
    • Evaluates the final model's performance with precision, recall, F1-score, ROC curve and confusion matrix.
  • Random Data Generation and Prediction:
    • Implements a function to generate random transaction data and make predictions using the model.
    • Saves the predicted data in an Excel sheet for review.
  • Pipeline Creation:
    • Creates a streamlined pipeline for scaling the data and making predictions efficiently.

Technical Details

  • Programming Language: Python 3.11.
  • Libraries/Modules Used:
    • pandas: For data manipulation and analysis.
    • numpy: For numerical computations.
    • scikit-learn: For model training, evaluation, and pipeline creation.
    • imbalanced-learn: For handling imbalanced datasets.
    • openpyxl: For working with Excel files.
  • Data Storage: CSV (`creditcard.csv`), Excel Sheet.
  • Output: Trained model, performance metrics, predicted data in Excel.

How to Use

  1. Install Required Libraries:

    Ensure you have all the necessary libraries installed. Use pip to install them:

    pip install pandas numpy scikit-learn imbalanced-learn openpyxl
  2. Prepare the Data:

    Place the CSV data file (`creditcard.csv`) in the same directory as your Python script or specify the correct path.

  3. Run the Script:

    Execute the Python script or Jupyter Notebook containing the project code.

  4. Analyze the Output:

    Review the output on the console which will contain model performance metrics and the Excel file containing the predicted results from random generated data
