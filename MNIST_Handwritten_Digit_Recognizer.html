MNIST Digit Recognition

Project Overview

This project implements a Convolutional Neural Network (CNN) to recognize handwritten digits from the MNIST dataset. The project consists of two main parts: a training script (mnist.ipynb) and a testing script (testing_mnist.ipynb). The training script loads the MNIST dataset, builds and trains the CNN, and saves the trained model. The testing script loads this saved model and uses it to predict the digit in a custom image.

Core Functionalities

  • Training Script (mnist.ipynb):
    • Data Loading and Preprocessing:
      • Loads the MNIST dataset using tensorflow.keras.datasets.mnist.
      • Normalizes pixel values to a range between 0 and 1.
      • Reshapes data from 2D images to 4D tensors with shape (num_samples, 28, 28, 1).
    • CNN Model Definition:
      • Constructs a sequential CNN using TensorFlow and Keras.
      • CNN consists of two Conv2D layers, followed by MaxPooling2D layers, flattening layer, dense layers with ReLU activation and dropout and a softmax output layer for digit classification.
    • Model Compilation and Training:
      • Compiles the CNN using Adam optimizer, sparse categorical crossentropy loss and accuracy metrics.
      • Trains the model for 10 epochs using training data and validated with test data using batch size of 64.
    • Visualization and Model Saving:
      • Plots the training history to visualize accuracy and loss curves.
      • Saves the trained model to digit_recognition_model.h5.
  • Testing Script (testing_mnist.ipynb):
    • Model Loading and Prediction:
      • Loads the pre-trained model from the digit_recognition_model.h5 file.
      • Defines a predict_custom_image function to load, preprocess, and predict digits from custom images, displaying the predicted digit and confidence scores.

Technical Details

  • Programming Language: Python 3.11.
  • Libraries/Modules Used:
    • tensorflow and keras: Used for building, training, and using neural networks.
    • matplotlib: Used for plotting and data visualization.
    • numpy: Used for numerical computations and data manipulation.
    • PIL (Pillow): Used for image loading and preprocessing in custom image prediction.
  • Data Storage: MNIST Dataset and Saved Model File (`digit_recognition_model.h5`)
  • Output: Trained model, Plots of training progress and Digit Prediction

How to Use

  1. Install Required Libraries:

    Make sure you have all the necessary libraries installed. You can install them using pip:

    pip install tensorflow matplotlib numpy Pillow
  2. Prepare the Environment:

    Ensure you have the necessary environment setup with libraries like Tensorflow, keras matplotlib and numpy

  3. Run the Training Script:

    Execute the mnist.ipynb notebook to train the CNN model. This will save the model as digit_recognition_model.h5.

  4. Run the Testing Script:

    Execute the testing_mnist.ipynb notebook to test the loaded model. This will predict the digit in the custom image.

  5. Provide Image path:

    Ensure a valid image path is provided in the testing script so the model can read the image, predict the digit and show the confidence scores.
