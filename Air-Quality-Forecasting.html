Air Quality Forecasting using ML

Project Overview

This project focuses on forecasting air quality using supervised machine learning techniques. It uses the Facebook Prophet library to predict future air quality based on historical data.

Core Functionalities

  • Data Loading and Preprocessing:
    • Loads air quality data from a CSV file (`AirQualityUCI.csv`).
    • Correctly parses CSV file separated by semicolons and converts decimal notation.
    • Removes unwanted columns and rows containing only null values from the dataset.
    • Replaces missing values (represented by -200) with `NaN`.
    • Fills the NaN values using the mean of their respective columns.
  • Data Preparation:
    • Combines date and time information into a single column for time series analysis.
    • Converts the combined date and time column to datetime format.
    • Prepares the data in the required format for Prophet forecasting, with `ds` (datetime) and `y` (target variable) columns.
  • Forecasting with Prophet:
    • Trains a time series forecasting model using the Facebook Prophet library.
    • Generates future timestamps and uses the trained model to make forecasts.
    • Plots the forecast data, including trend, yearly, and weekly components.

Technical Details

  • Programming Language: Python 3.11.
  • Libraries/Modules Used:
    • pandas: Used for data manipulation and analysis, particularly with DataFrames.
    • numpy: Used for numerical computations, including handling `NaN` values.
    • prophet: Facebook’s Prophet library, used for time series forecasting.
  • Data Storage: CSV (`AirQualityUCI.csv`).
  • Output: Plots showcasing the forecasted data and its components.

How to Use

  1. Install Required Libraries:

    Make sure you have all the necessary libraries installed. You can install them using pip:

    pip install pandas numpy prophet
  2. Prepare the Data:

    Ensure the CSV data file (`AirQualityUCI.csv`) is in the same directory as your Python script or provide the correct file path.

    Ensure data in CSV file is seperated by ';' and decimal values are represented with ','

  3. Run the Script:

    Execute the Python script or Jupyter Notebook containing the provided code.

  4. Review the Output:

    Review the generated plots showing the air quality forecast. The forecast will include trend, yearly, and weekly patterns.

In Which Sector is it Used

  • Environmental Monitoring: For governments and environmental agencies to monitor and predict air quality, helping in planning and interventions.
  • Urban Planning: Useful for urban planners to understand the environmental impact of development projects and plan for better air quality.
  • Public Health: For public health officials to forecast health risks associated with poor air quality, allowing them to issue timely warnings and advice.
